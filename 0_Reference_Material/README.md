# 📚 Reference Material

This folder includes datasheets, PCB fabrication notes, and related hardware documentation for the **Measurement Network Gateway (MNG)** project.

| File | Description |
|------|--------------|
| `BaseSystem.pdf` | Overview of base embedded system and connections |
| `Fabr_48747P01.pdf` | PCB fabrication and Gerber layer documentation |

These documents serve as technical references for hardware selection, interface design, and validation.
